## Icones

Para definir o tamanho do icone você pode aplicar a classe nele:

| valor             | tamanho |
| ----------------- | ------- |
| icon-size-small   | 16px    |
| icon-size-medium  | 24px    |
| icon-size-large   | 32px    |
| icon-size-x-large | 40px    |
| icon-size-giga    | 48px    |

Exemplo: 

```html dark
<Icon class="icon-size-small">
```
